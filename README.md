REVA Base
REVA Base is the foundational merchant engine of the REVA ecosystem.
It provides:

•	Inventory state management
•	Smart POS functionality
•	Margin and stock visibility
•	Ledger-aware transaction tracking

REVA Base is designed as a modular system that will integrate into REVA Connect, enabling marketplace expansion, payment terminals, and carrier integrations.
